# SERVICES.md

Every external account and service used by Todoclaw: what each does, how they connect, which keys live where, and links to dashboards.

Services: Supabase, Vercel, Anthropic Console, GitHub (Actions, Dependabot, CodeQL, secret scanning), Sentry.

Also contains the security incident runbook: what to do when a Dependabot/Socket/Sentry alert fires, and how to rotate a compromised key.

> This file is filled in during Stage 1 (cloud accounts created) and Stage 2 (Sentry, CI wired up).
